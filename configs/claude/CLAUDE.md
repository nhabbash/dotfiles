## Context files (load on demand)

- `~/.config/agents/design-principles.md` — Software design philosophy: mechanism vs policy, stable/fast elements, composability. Read when making architectural decisions, designing interfaces, or evaluating whether something belongs in a core or at an edge.
- `~/.config/agents/monday-sprint-board.md` — Monday.com sprint board config (board/column IDs, API patterns). Read when asked to create, read, update, or manage sprint tasks, tickets, or anything related to the Campaigns team board.
- `~/.config/agents/local-dev-tips.md` — Monday Mirror GraphQL testing, port discovery, auth headers for local dev.
- `~/.config/agents/slack.md` — Slack workspace map: team members (IDs), channels (IDs), messaging conventions, and tool usage. Read when sending messages, searching Slack, posting PRs, writing standups, or any Slack interaction.
