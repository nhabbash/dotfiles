# Work Profile Extensions

Additional context available on the work machine. These files are not in the dotfiles repo — they live on the work machine only.

## Context files (load on demand)

- `~/.config/agents/slack.md` — Slack workspace map: team members (IDs), channels (IDs), messaging conventions, tool usage. Load when sending messages, searching Slack, posting PRs, or writing standups.
- `~/.config/agents/monday-sprint-board.md` — Monday.com sprint board config (board/column IDs, API patterns). Load when creating, reading, or updating sprint tasks.
- `~/.config/agents/local-dev-tips.md` — Work-specific local dev: service ports, auth headers, GraphQL testing patterns, internal tooling.
